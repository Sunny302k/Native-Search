# NOTE — Merge Value

## 1. What is Merge Value (from UI description)
- Merge Value allows combining multiple filter option values into one single displayed value
- Used to simplify filter options for storefront users

---

## 2. Merge Value Overview Page

### Description block
- Explains that Merge Value helps group similar filter values
- Used to reduce the number of filter options shown on storefront

### Main action
- Button: Create new merge value

---

## 3. Create Merge Value Flow

### Step 1 — Select Filter
- User selects which filter the merge value applies to
- Filter type / attribute selection is required before continuing
- Action buttons:
  - Cancel
  - Continue / Next

---

### Step 2 — Select Values
- Section displays list of available filter option values
- User can select multiple values
- Selected values are grouped into one merge set
- Option to add more values

---

### Step 3 — Configure Merged Value
- Input field:
  - Merged value name (display label)
- UI shows list of selected original values
- Action buttons:
  - Back
  - Save

---

## 4. Merge Value List

Each merge value item shows:
- Applied filter
- Merged value name
- Original values included in the merge
- Status toggle (enable / disable)
- Action icons:
  - Edit
  - Delete

---

## 5. Edit Merge Value
- Same steps as Create flow
- Existing data is pre-filled:
  - Selected filter
  - Selected values
  - Merged value name

---

## 6. Delete Merge Value

### Confirmation dialog
- Title: Delete merge value?
- Warning message indicates deletion cannot be undone
- Buttons:
  - Cancel
  - Delete

### After delete
- Success message is shown
- Merge value is removed from the list

---

## 7. Notes / Observations
- Merge Value only affects display grouping of filter values
- No effect is shown in UI for sorting or priority
- Merge Value is managed independently from Filter Cache
