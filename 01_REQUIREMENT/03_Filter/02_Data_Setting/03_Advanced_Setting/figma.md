# Figma Reference — Advanced Settings

## Feature scope
- Feature: Advanced Settings
- Parent feature: Filter
- Scope: Display & behavior settings applied to Filter on storefront

## Entry point
- Navigation: Filter → Advanced Settings

## Screens & states

### 1. Advanced Settings Page
- Left panel: List of advanced toggle settings
- Right panel: Live preview of Filter (Desktop / Mobile)
- Top right action: Save Changes

---

## Settings shown in Advanced Settings

### A. Show smart filter search
- Toggle control
- Applies to filter nodes using list / grid selection type
- When ON:
  - A search input appears at the top of filter options
- Preview shown for:
  - Desktop
  - Mobile

---

### B. Show product count
- Toggle control
- When ON:
  - Product count is displayed next to each filter option
- When OFF:
  - Product count is hidden

---

### C. Show Refine by block
- Toggle control
- When ON:
  - A "Refine by" block is displayed
  - Shows selected filters
  - Each selected filter can be removed individually
  - Clear all option is available
- Preview shown in Desktop & Mobile

---

### D. Show all irrelevant values (product count = 0)
- Toggle control
- Default state: OFF
- When ON:
  - Filter options with product count = 0 are displayed
- When OFF:
  - Filter options with product count = 0 are hidden

---

### E. Shorten URL when selecting multiple filter option values
- Toggle control
- Affects storefront URL structure when multiple filter values are selected
- Preview includes comparison:
  - Long URL format
  - Shortened / SEO-friendly URL format

---

### F. Hide filter options with only one filter option value
- Toggle control
- When ON:
  - Filter options with product count = 1 are hidden

---

### G. Change label of Refine By button on mobile when tapping on it
- Toggle control
- Applies to Mobile view only
- Changes label behavior when user taps Refine By

---

### H. URL Structure
- Informational section
- Shows examples:
  - Old URL format
  - New URL format
  - SEO-friendly URL format
